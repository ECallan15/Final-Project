# Churn Analysis Final
This project predicts customer churn using a structured machine learning pipeline.  
The code is modular and reproducible, with all production-ready code in `src/`.

## Installation
Install required Python packages:
```
pip install -r requirements.txt
```
## Run
In the project folder, run:
```
python main.py
```
This will:
- Load and preprocess data  
- Generate features  
- Train models (k-NN & Decision Tree)  
- Evaluate metrics  
- Create `submission.csv`
